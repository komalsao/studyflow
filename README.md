# StudyFlow

StudyFlow is an AI-powered study companion for students designed to help them organize their learning, understand concepts more effectively, and build consistent study habits.

The goal of this project is to create a clean, intuitive platform where students can upload their study material and receive AI-powered assistance through summaries, flashcards, quizzes, personalized study plans, and an interactive study companion named Lumi.

This project is currently under active development.

---

## Current Features

- Responsive landing page
- Multi-page navigation using React Router
- Welcome page with Guest Mode
- UI designed around Lumi, the AI study companion
- Clean and minimal user interface
